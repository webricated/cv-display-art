# Dr. Anirvan Mukherjee — Academic Portfolio

A responsive academic portfolio website built with React, Vite, TypeScript, and Tailwind CSS. It showcases education, research experience, publications, conferences, and expertise for Dr. Anirvan Mukherjee.

## Deployment

This project is configured for GitHub Pages via the workflow in `.github/workflows/deploy.yml`. The site is deployed at:

`https://webricated.github.io/anirvan-mukherjee/`

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
